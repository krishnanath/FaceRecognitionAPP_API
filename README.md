# face-recognition-api
1. clone this repo
2. Run ```npm install```
3. Run ```npm start```

* use postgreSQL instead of mySQL for this code base.

**Packages Installed** 
* ``` $ npm install express```
* ```$ npm install body-parser```
* ```$ npm install bcrypt-nodejs```
* ```$ npm install corn```
* ```$ npm install knex --save```
